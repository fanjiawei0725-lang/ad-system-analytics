# Deliverables

本目录用于长期归档可独立查看或交付的文件，包括 HTML、文档、图片和配套资源。

## 目录约定

```text
deliverables/
└── <year>/
    └── <topic>/
        ├── index.html
        ├── README.md
        └── assets/
```

- 按年份和主题归档，主题目录使用小写英文与连字符。
- 可直接浏览的 HTML 统一命名为 `index.html`。
- 图片、脚本等配套文件放在同级 `assets/` 中，并使用相对路径引用。
- 同一主题的后续版本优先在原目录维护；需要保留历史版本时，再增加 `archive/`。
