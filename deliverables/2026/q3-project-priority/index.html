增编触发条件

不是“项目多”，而是“关键角色持续不足”

  • 压缩 Q2、Q3、Q4 范围后，投放最低阵容仍无法连续保障 10 周。
  • 同一稀缺角色被两个一级里程碑同时占用，且无法通过顺序调整化解。
  • 补人顺序按真实瓶颈决定：数据迁移 / 口径 → 数据；渠道集成 → 后端；素材与 Agent 并行 → AI。

立项清单与 Q3 边界

所有项目都给出明确的“做什么”和“不做什么”，避免高价值方向被无限扩张。
